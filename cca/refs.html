Reference Links

Primary sources for the Claude Certified Architect Foundation exam. Every topic tested on the exam traces back to one of these official documents. Bookmark the ones you find yourself returning to.

ⓘ
Exam strategy: The CCA-F tests your ability to apply these docs — not memorise them. Focus on understanding why each design decision exists. The highest-yield sections are: Messages API (stop_reason), Tool Use (descriptions), Hooks (deterministic enforcement), and CLAUDE.md hierarchy (project vs user scope).
